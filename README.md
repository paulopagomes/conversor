
# Gerar EXE no Windows sem instalar nada no seu PC (via GitHub Actions)

## Como usar
1) Crie um repositório no GitHub (público ou privado)
2) Envie TODOS os arquivos desta pasta para o repositório
3) Vá em: Actions -> Build Windows EXE (Full) -> Run workflow
4) Quando terminar, baixe o Artifact: ConversorPDFImagem-EXE

## Observações
- O EXE gerado não precisa de Python instalado para rodar.
- Office->PDF no Windows só funciona se você tiver LibreOffice instalado e acessível no PATH (opcional).
